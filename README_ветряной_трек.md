# Ветряной трек ExCentro — что и куда добавить

## 1. Файл презентации

`excentro-wind-ru.html` → загрузить в корень репозитория `excentro-vm`
(GitHub → Add file → Upload files → Commit changes).

Адрес после публикации:
**https://pokerface-hub.github.io/excentro-vm/excentro-wind-ru.html**

Ссылки внутри уже настроены:
- «Опросный лист → расчёт» → `excentro-calculator.html` (относительная, работает в той же папке);
- Telegram-кнопки → `t.me/ExCentroAssistantBot?start=wind_ru` и `?start=nda_wind`
  (боту нужно добавить обработку этих параметров — см. п. 4).

## 2. Карточка на index.html

Добавить рядом с существующими карточками презентаций (RU/EN/ES) блок-ссылку:

```html
<a href="excentro-wind-ru.html" class="card"> <!-- класс подставить как у соседних карточек -->
  <div class="card-tag">НОВОЕ · ОТРАСЛЕВОЙ ТРЕК</div>
  <h3>Ветроэнергетика (ВЭУ)</h3>
  <p>Комплексное предложение: приводы поворота и питча (ЭЦ-ПЗР) — сегодня,
     мультипликатор ВМ — перспектива. 10 слайдов.</p>
</a>
```

Точную разметку подгоните под структуру ваших карточек — либо пришлите мне
исходник index.html, верну готовый файл с интегрированной карточкой в вашем стиле.

## 4. bot.py — минимальный патч (описание)

- deep-link `start=wind_ru` → приветствие ветряного трека + отправка PDF-тизера
  (файл `Тизер_ЭЦ-модуль_ВЭУ_неконфиденциально.pdf` положить рядом с bot.py
  или в репо) + 3 вопроса квалификации (роль: производитель/оператор/институт;
  парк/мощность; интерес: yaw-pitch / трансмиссия / оба);
- deep-link `start=rosatom` → тот же сценарий, но в журнал лидов пишется
  источник «Росатом-письмо» (для QR из письма Локшину);
- `start=nda_wind` → отправка проекта NDA + запись лида с пометкой «NDA-запрос · ВЭУ»;
- в строку лида добавить колонку «Трек» (общепром / ВЭУ) — фильтровать в таблице.

Пришлите текущий bot.py (или дайте знать — заберу из репо сама) — верну готовый
патченный файл.
