Готовый комплект для публикации на GitHub Pages.

## Состав

| Файл | Назначение |
|------|-----------|
| `index.html` | Лендинг с выбором языка + кнопка калькулятора (стартовая страница) |

Все ссылки между файлами — относительные. Файлы должны лежать в **одной папке / одном репозитории**.

## Как опубликовать на GitHub Pages (5 минут)

### Вариант А — через сайт GitHub (с телефона или компьютера)

1. Зайди на **github.com**, войди под `pokerface-hub`.
2. Нажми **New repository** (зелёная кнопка). 
   - Имя: например `excentro-vm` 
   - Тип: **Public** 
   - Поставь галочку **Add a README** → **Create repository**.
3. В репозитории нажми **Add file → Upload files**.
4. Перетащи СЮДА все файлы из этого комплекта (`index.html`, три `excentro-vm660-*.html`, `excentro-calculator.html`, и PDF при желании).
5. Внизу нажми **Commit changes**.
6. Открой вкладку **Settings → Pages** (слева в меню).
7. В разделе **Build and deployment → Source** выбери **Deploy from a branch**, 
   Branch: **main**, папка **/ (root)** → **Save**.
8. Через 1–2 минуты вверху появится ссылка вида: 
   **`https://pokerface-hub.github.io/excentro-vm/`**

Готово. По этой ссылке открывается лендинг, оттуда — презентации и калькулятор. Кнопки, расчёт и переходы работают (в отличие от предпросмотра Google Drive).

### Вариант Б — если уже есть репозиторий vm-selector

Можно положить файлы в подпапку существующего репозитория `vm-selector`, тогда адрес будет 
`https://pokerface-hub.github.io/vm-selector/excentro/` — но проще завести отдельный репозиторий (Вариант А).

## Важно

- **Не открывай файлы в предпросмотре Google Drive / облака** — там JavaScript и переходы между файлами заблокированы (отсюда «404» и «расчёт не идёт»). Работает только: GitHub Pages, либо скачать всё в одну папку и открыть в браузере.
- Для калькулятора нужен интернет только для загрузки шрифтов (есть системные фолбэки — без интернета тоже читаемо).
